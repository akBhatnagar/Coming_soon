# Coming Soon | Akshay Bhatnagar

A minimal, animated "Coming Soon" landing page.

**Live:** [comingsoon.akshaybhatnagar.in](https://comingsoon.akshaybhatnagar.in)

## Features

- Clean, centered design with animated elements
- Responsive layout for all screen sizes
- Built with React and Tailwind CSS

## Quick Start

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

| Layer     | Technology   | Purpose                   |
|-----------|-------------|---------------------------|
| Framework | React 19    | Component-based UI        |
| Styling   | Tailwind CSS| Utility-first CSS         |
| Build     | CRA         | Create React App toolchain|
